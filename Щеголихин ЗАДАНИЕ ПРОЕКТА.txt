Задание проекта
Название проекта: "Телеграмм-бот для управления и анализа данных"

Цель проекта: Создание универсального телеграмм-бота, который позволяет пользователям получать информацию о погоде, создавать события, анализировать данные, обрабатывать изображения и проводить платежные операции.

Используемые технологии:
- Python 3
- Библиотека python-telegram-bot для разработки телеграмм-бота
- API для получения погоды (например, OpenWeatherMap API)
- Библиотека PIL для обработки изображений
- SQLAlchemy для работы с базой данных (SQLite)
- Модуль hashlib для безопасной аутентификации пользователей

Основные функции:
1. Получение погоды по запросу пользователя.
2. Создание событий и добавление их в календарь.
3. Анализ числовых данных, введенных пользователем.
4. Обработка изображений (например, преобразование в черно-белое изображение).
5. Проведение платежных операций.
